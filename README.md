# tofuref

A terminal application to browse OpenTofu provider documentation.

![Screenshot](https://github.com/djetelina/tofuref/blob/main/screenshots/welcome.png?raw=true)

## Installation

```bash
pipx install tofuref
```

## Usage

Run the application:

```bash
tofuref
```

## Upgrade

```bash
pipx upgrade tofuref
```
